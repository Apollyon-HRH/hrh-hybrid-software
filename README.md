# HRH Hybrid Software – Team Apollyon

Sistema híbrido alternativo, ferramentas de integração e scripts Lua para a categoria Hypercar Rally Hybrid (HRH), conforme Parecer Técnico PT-2026-001.

## Módulos

- **Lua scripts** – sistema híbrido (MGU-KF+KR), aquisição de dados, controlo de tração, travagem regenerativa, pit limiter, ABS off-road.
- **tools/pc_hybrid_injector.py** – conversão de ficheiros `.pc` do Automation para arquitetura híbrida.
- **ci/** – testes automatizados com BeamNG.tech (Apêndice C12).
